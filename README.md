# Nethravathi Basin — Rainfall Threshold Mapping and GNN Flood Prediction

End-to-end hydrological modelling and machine learning workflow for 
flood threshold mapping in the upper Nethravathi River basin, 
Karnataka, India. Simulates the August 2018 SW monsoon flood event 
using a physics-based SCS-CN rainfall-runoff model calibrated against 
35 years of observed discharge, derives rainfall threshold maps from 
seven scenario simulations, and trains a Graph Convolutional Network 
(GCN) to predict flood depth from terrain and rainfall features.

## Key results

| Component | Result |
|---|---|
| Calibration year | 2014 (year-matched, within discharge record) |
| Calibration PBIAS | −15.7% (satisfactory, within ±20% threshold) |
| NSE proxy | 0.975 |
| 1× event flood extent | 12.3% of basin flooded (depth > 0.1 m) |
| 2× scenario flood extent | 22.0% of basin flooded |
| Threshold map range | 0 – 1,252 mm rainfall to initiate flooding |
| GCN R² | 0.640 |
| GCN RMSE | 0.040 m |
| GCN MAE | 0.023 m |

## Study area

- **Basin:** Upper Nethravathi sub-basin, Karnataka, India
- **AOI:** 75.55°E–76.05°E, 12.80°N–13.20°N (~800 km²)
- **Full basin at Bantwal gauge:** ~3,657 km²
- **Grid resolution:** 30 m (UTM Zone 43N, EPSG:32643)
- **Flood event simulated:** August 8–18, 2018 (209 mm basin-mean)
- **Calibration event:** June–September 2014 (1,358 mm)

## Model summary

### Hydrological model
- **Method:** SCS-CN rainfall-runoff with HAND-based depth surrogate
- **Calibration:** Year-matched 2014, area-scaled observed discharge
- **Parameters:** CN multiplier ×1.50, Ia ratio = 0.05
- **Flood depth:** `depth = (Q/1000) × exp(−HAND/10) × exp(−slope/25) × (1 + 2×facc_norm)`

### Graph Convolutional Network
- **Architecture:** Input(6) → GCNConv(64) → GCNConv(32) → Linear(1)
- **Nodes:** 72,268 (every 6th pixel, ~180 m spacing)
- **Edges:** 578,144 (k=8 nearest neighbours)
- **Parameters:** 2,561
- **Training:** 200 epochs, Adam lr=0.001, MSE loss

## Limitations

- Sub-basin scope: 800 km² covers 22% of full Bantwal catchment
- No channel routing — travel time and flood-wave propagation absent
- CHIRPS daily resolution smooths sub-daily orographic peaks
- GCN trained on one simulated event only
